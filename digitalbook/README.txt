# ドキュメント

1. セットアップと実行の手順
　VSC等Pythonの実行環境においてmain.pyのスクリプトを実行してください。

2. プロジェクトにおける重要な設計やその設計理由
- 対話型のMenuにし、常にユーザーに処理を問いかける形にした（While True文）
　‐　理由：アプリが止まってしまうと、ユーザーは次にどうすればいいのか分からなくなる為、常に選択肢を提示して能動的にアクションを呼び掛けた。
- エッジケースに対応する為、各々の処理についてIf文を用いた
　- 理由：ユーザーが誤入力する事を想定し、事実と次のアクションを明確に提示する為に設計した。

3. このツールまたはサービスの使い方の説明 (ユーザー向けの説明)
- 起動すると以下のMenu画面が現れますので、行いたい処理に応じて、番号を入力してください。

    "=================================================\n"+
    "Welcome to your personal books digital library!\n"+
    "=================================================\n"+
    "1: Add a Book\n"+
    "2: Edit a Book\n"+
    "3: Search for Books\n"+
    "4: Delete a Book\n"+
    "5: View Library Stats\n"+
    "6: Exit app"

- 1. 本を追加したい場合
  - Menu画面で1（半角英数）を入力ください。
　- 初めに、本のタイトルを入力してください。
　- 次に、既読か未読かを入力してください。（既読の場合は1を、未読の場合は2を半角英数で入力してください。）

- 2. 本の内容を編集したい場合
　- Menu画面で2（半角英数）を入力ください。
　- 編集したい本のID（半角英数）を入力してください。（本のIDが不明な場合はMenuの　3: Search for Books　から検索して確認してください。）

- 3. タイトルから本を検索し、登録内容を閲覧したい場合
　- Menu画面で3（半角英数）を入力ください。
　- 本のタイトルに含まれる文字を入力してください。
　- 本のタイトルに入力された文字が含まれている場合、本のID、本のタイトル、既読か未読（既読の場合はYes、未読の場合はNo）かの情報が表示されます。

- 4. 本を削除したい場合
　- Menu画面で4（半角英数）を入力ください。
　- 削除したい本のIDを入力してください。（本のIDが不明な場合はMenuの　3: Search for Books　から検索して確認してください。）
　- 削除された本のIDとタイトルが表示されます。

- 5. 現在の本の登録数、既読の本の数、未読の本の数を確認したい場合
　- Menu画面で5（半角英数）を入力ください。
　- このライブラリーに登録されている、現在の本の登録数、既読の本の数、未読の本の数が表示されます。

- 6. このアプリから退出する場合
　- Menu画面で6（半角英数）を入力ください。(退出すると、登録された本の情報は全て削除されます。)
